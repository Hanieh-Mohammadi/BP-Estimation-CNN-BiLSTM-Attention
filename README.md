## Overview

The proposed model estimates continuous arterial blood pressure (ABP) waveforms from photoplethysmography (PPG) signals using:
- A CNN-based feature extractor  
- BiLSTM layers for temporal modeling  
- Attention mechanism to emphasize informative waveform regions

---

## Dataset

Experiments were conducted using the public **MIMIC-II waveform database**.
